Dash — библиотека пользовательского интерфейса для создания аналитических веб-приложений. 
Те, кто использует Python для анализа данных, исследования данных, визуализации, моделирования, управления приборами и отчетности, 
сразу же найдут применение для Dash.

Здесь приведены простейшие примеры (PyCharm)

Для запуска приложений необходимо установить пакеты dash, pandas, dash_html_component, dash_canvas
